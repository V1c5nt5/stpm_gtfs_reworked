STPM GTFS Map App

Aplicación cartográfica para explorar recorridos, paraderos y trazados GTFS. Los datos se descargan desde GitHub al cargar.

Los archivos pesados no vienen dentro de esta web. Se leen desde https://github.com/V1c5nt5/stpm_gtfs/tree/main/data. También puedes cargar archivos manuales.
